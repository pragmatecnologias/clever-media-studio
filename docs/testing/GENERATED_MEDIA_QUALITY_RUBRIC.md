# Generated Media Quality Rubric

Score each campaign 1-5 per category. Target average: >= 3.5.

## 1. Message Clarity
*Does the media package communicate the intended message clearly?*
- 5: Core message instantly clear across all assets
- 4: Clear but one asset slightly off-topic
- 3: Message present but takes effort to find
- 2: Different assets say different things
- 1: No clear message — assets feel random

## 2. Spiritual Appropriateness
*Is the content appropriate for church/ministerial context?*
- 5: Naturally pastoral, theologically sound
- 4: Appropriate but one element slightly off-tone
- 3: Acceptable but misses nuance (e.g., sales-y CTA)
- 2: Language/imagery that would raise eyebrows
- 1: Clearly inappropriate for church context

## 3. Visual Quality
*Do slides and social images look professional?*
- 5: Clean, modern, readable — comparable to professional design
- 4: Good with minor issues (font size, slight contrast)
- 3: Acceptable but generic — looks template-generated
- 2: Visibly rough — text overlaps, poor alignment
- 1: Broken — missing images, unreadable text

## 4. Slide Usefulness
*Would a pastor use these slides in a service?*
- 5: Ready to use — headlines, body, structure all work
- 4: Usable with minor edits (one headline, verse ref)
- 3: Good starting point — structure works, needs rewriting
- 2: Skeleton only — right count, placeholder content
- 1: Not usable — wrong structure, irrelevant

## 5. Social Invitation Effectiveness
*Do social assets effectively invite, inform, or inspire?*
- 5: Compelling — clear CTA, correct details, platform-appropriate
- 4: Good — CTA present but could be stronger
- 3: Basic — details there but lacks warmth/urgency
- 2: Weak — missing CTA, wrong sizes, irrelevant imagery
- 1: Not usable — incorrect details, broken layout

## 6. Caption Naturalness
*Do captions sound like a real person wrote them?*
- 5: Natural and warm — reads like staff wrote it
- 4: Mostly natural — one or two templated phrases
- 3: Functional — gets info across but feels formulaic
- 2: Stiff — clearly template-generated, repetitive
- 1: Robotic — contains prompt language or metadata

## 7. Brand and Event Correctness
*Are all facts and details correct?*
- 5: All correct — date, time, timezone, location, church, CTA
- 4: One minor detail missing (timezone label)
- 3: One key detail wrong (wrong day or time format)
- 2: Multiple errors — date AND location wrong
- 1: Completely wrong or fabricated

## 8. Export Usability
*Is the exported ZIP complete and usable?*
- 5: Complete — PPTX, PDF, PNGs, JPGs, captions (3 formats), manifest
- 4: Most formats — missing one non-critical format
- 3: Core formats — missing important ones (no PDF or no images)
- 2: Skeleton — JSON/text only, no media files
- 1: Broken — empty ZIP, corrupted files
